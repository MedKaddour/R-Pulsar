Installation

Dependencies

R-Pulsar is written in Java. For storing data, R-Pulsar relies on RocksDB, an embedded key-value database.

Compiling the platform

Compile using maven:

mvn clean package

R-Pulser consumer/producer examples

A set of examples are provided within the R-Pulsar archive, including configuration and property files:

Hello World

A simple example of communication between a data producer, a data consumer and a Rendez-vous Point(RP). Producer and consumer start by creating profiles. Then the RP successfully matches the profiles, resulting in a stream of messages from one to the other.

Storm Window

Producer streams message composes of sentences. Consumer relies on Apache Storm to count the occurences of words using window processing based on Apache Storm.

Rules

This example declares two consumers located respectively on Edge and Cloud resources. Based on the content of each data product within the stream, computations are triggered either at the edge or in the cloud.

Master/Slave Rendez-vous Points

This examples illustrates the ability to work with multiple RPs within across a federation of resources.
